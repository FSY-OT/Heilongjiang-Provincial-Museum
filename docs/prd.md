# 需求文档

## 1. 应用概述

### 1.1 应用名称
黑龙江博物馆信息展览大屏

### 1.2 应用描述
面向黑龙江博物馆管理层及运营人员的数据可视化大屏系统，实时展示博物馆核心运营指标、客流分析、馆藏管理、设备监控等多维度数据，采用科技感可视化风格，支持大屏展示。

## 2. 用户与使用场景

### 2.1 目标用户
- 博物馆管理层：查看整体运营状况
- 运营人员：监控实时数据与异常情况
- 参观者服务中心：了解客流与服务状态

### 2.2 核心使用场景
- 管理中心大屏实时展示博物馆运营数据
- 运营会议数据汇报与分析
- 日常运营监控与决策支持

## 3. 页面结构与功能说明

### 3.1 页面结构

```
黑龙江博物馆信息展览大屏
├── 顶部指标栏
├── 左侧面板
│   ├── 文创·电商·流量模块
│   ├── 馆藏·文物·保护模块
│   └── 历史对比与预警模块
├── 中间背景展示区
└── 右侧面板
    ├── 客流·安防·服务模块
    ├── 设备·能源·应急模块
    └── 滚动信息模块
```

### 3.2 功能说明

#### 3.2.1 顶部指标栏（核心战略指标）

展示博物馆核心KPI数据：
- 馆藏总资产：显示数值
- 今日总预约：显示数值
- 年累计客流：显示数值
- 安全无故障：显示天数
- 环境达标率：显示百分比
- 历史最高客流：显示数值及日期

#### 3.2.2 左侧面板

**文创·电商·流量模块**
- 文创产品总销售额：显示金额数值
- 爆款单品：展示铜坐龙冰箱贴及销售数据
- 线上访问UV：显示访问量
- 热门排行占比：使用饼图或柱状图展示猛犸象玩偶、冰城雪景书签等产品占比
- 游客画像：
  - 年龄构成：使用柱状图或饼图展示不同年龄段占比
  - 省外游客比例：显示百分比

**馆藏·文物·保护模块**
- 馆藏分类统计：
  - 历史文物：显示总数及展出数量
  - 自然标本：显示总数及展出数量
  - 少数民族文物：显示总数及展出数量
  - 艺术藏品：显示总数及展出数量
  - 待修复文物：显示数量
  - 数字藏品：显示数量
- 一级文物数：显示总数
- 一级文物备注：列出松花江猛犸象化石、赫哲族鱼皮衣等重点文物名称

**历史对比与预警模块**
- 入馆人次：显示今日数值、同比变化、环比变化
- 文创收入：显示今日数值、同比变化、环比变化
- 团队预约：显示今日数值、同比变化、环比变化
- 讲解场次：显示今日数值、同比变化、环比变化
- 馆内停留时长：显示今日数值、同比变化、环比变化

使用折线图或趋势图展示历史对比数据。

#### 3.2.3 中间背景展示区

- 展示背景图片：https://miaoda-conversation-file.cdn.bcebos.com/user-7wvoy7ho0glc/app-buovwyuyvzlt/20260524/屏幕截图 2026-05-24 110544.png
- 背景图上可叠加半透明遮罩效果，与深蓝色系UI风格契合
- 可在背景图上标注关键点位或核心视觉元素

#### 3.2.4 右侧面板

**客流·安防·服务模块**
- 客流高峰：显示时间段（10:30-11:30）及峰值人数，使用折线图展示全天客流趋势
- 讲解服务：显示今日讲解场次（45场）
- 智慧安防：显示监控点总数（368个）及在线状态
- 展厅拥堵排行：使用柱状图展示自然展厅、冰雪文化体验区等展厅的拥堵程度
- 环境监控：显示温度（22.5℃）、湿度（51%）

**设备·能源·应急模块**
- 设备使用情况：
  - 直饮水机：显示使用次数及故障报修数
  - 轮椅/童车：显示借用数量及故障报修数
  - 语音导览器：显示借用数量及故障报修数
- 公共能耗：显示今日用电量（1,207 kWh）

**滚动信息模块**
- 实时滚动展示博物馆资讯：
  - 新展预告
  - 安全检查通报
  - 荣誉通报
  - 社教活动通知
  - 其他实时资讯

## 4. 业务规则与逻辑

### 4.1 数据更新规则
- 核心战略指标、客流数据、环境监控数据实时更新
- 历史对比数据每日更新同比、环比数值
- 滚动信息根据后端推送实时更新

### 4.2 数据展示规则
- 数值类数据保留小数点后1位（如温度、湿度）
- 金额类数据使用千分位分隔符
- 百分比数据保留小数点后1位
- 同比/环比数据使用「↑」「↓」符号标识增长或下降趋势

### 4.3 图表展示规则
- 折线图：用于展示客流趋势、历史对比数据
- 柱状图：用于展示展厅拥堵排行、年龄构成
- 饼图：用于展示热门排行占比、游客画像
- 雷达图：可用于多维度数据对比分析

### 4.4 UI风格规则
- 整体采用深蓝色系配色方案
- 数据面板使用半透明遮罩效果
- 图表配色与整体风格保持一致
- 字体清晰易读，适配大屏展示

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 数据源异常或无数据 | 显示「--」或「数据加载中」 |
| 设备故障报修数为0 | 显示「0」 |
| 环境监控数据超出正常范围 | 数值标红或闪烁提示 |
| 滚动信息为空 | 显示「暂无最新资讯」 |
| 背景图片加载失败 | 显示纯色背景或默认占位图 |

1. 打开大屏页面，页面完整加载，显示背景图片（https://miaoda-conversation-file.cdn.bcebos.com/user-7wvoy7ho0glc/app-buovwyuyvzlt/20260524/屏幕截图 2026-05-24 110544.png）及深蓝色系UI风格
2. 顶部指标栏正确显示馆藏总资产、今日总预约、年累计客流、安全无故障、环境达标率、历史最高客流6项核心KPI数据
3. 左侧面板正确显示文创·电商·流量、馆藏·文物·保护、历史对比与预警三个模块的完整数据及图表
4. 右侧面板正确显示客流·安防·服务、设备·能源·应急、滚动信息三个模块的完整数据及图表
5. 中间背景展示区正确显示背景图片，与左右面板布局协调
6. 所有图表（折线图、柱状图、饼图等）正确渲染，数据准确展示
7. 滚动信息模块正常滚动播放新展预告、安全检查、荣誉通报、社教活动等资讯
